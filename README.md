# Pedido API

![Testes](https://github.com/lorrainenatielly/pedido-api/actions/workflows/node.yml/badge.svg)

## Sobre o projeto

Este projeto foi desenvolvido como uma simulação de uma solução para uma empresa fictícia que precisava melhorar o processo de cálculo de pedidos. O objetivo foi criar uma função capaz de calcular o valor final de um pedido aplicando descontos automaticamente, reduzindo erros manuais e aumentando a confiabilidade do processo.

A aplicação foi desenvolvida em Node.js, utilizando testes automatizados com Jest e integração contínua através do GitHub Actions.

---

## Problema enfrentado

A empresa simulada realizava cálculos de pedidos manualmente, o que poderia causar erros no valor final das compras e dificultar a manutenção do sistema. 

Para solucionar esse problema, foi criada uma função automatizada responsável por calcular o valor final de pedidos com aplicação de descontos percentuais, garantindo maior precisão e facilidade de testes.

## Uso de Inteligência Artificial no desenvolvimento

A Inteligência Artificial vem transformando o desenvolvimento de software ao auxiliar equipes na criação de códigos, testes e melhorias de produtividade.

O GitHub Copilot atua como um assistente de programação capaz de sugerir códigos a partir de comandos escritos pelo desenvolvedor. Ele pode auxiliar na criação de funções, encontrar possíveis melhorias e gerar testes automatizados.

Neste projeto, o GitHub Copilot foi utilizado para auxiliar na criação da função principal de negócio e na elaboração dos testes automatizados.

A utilização de IA no desenvolvimento contribui para:
- aumento da produtividade;
- redução do tempo de implementação;
- auxílio na identificação de erros;
- melhoria na qualidade do código.

Além disso, ferramentas como GitHub Actions permitem automatizar processos de integração contínua (CI/CD), executando testes automaticamente a cada alteração enviada ao repositório.
